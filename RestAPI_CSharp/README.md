# RestAPI_CSharp

This project replicates the Python REST API functionality in C# using ASP.NET Core Web API.

## Endpoints
- GET /api/rows
- GET /api/rows/{id}
- POST /api/rows
- PUT /api/rows/{id}
- DELETE /api/rows/{id}

## How to run
1. Install .NET SDK from https://dotnet.microsoft.com/download
2. Open a terminal in this folder
3. Run `dotnet run` to start the API
